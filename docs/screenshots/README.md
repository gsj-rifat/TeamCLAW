## Recording tips

1. Send a realistic message (decision + assigned todo), not `hello`.
2. Wait for the thread reply before switching to the dashboard.
3. Use **Activities** or **Global Search** to show persisted data.
4. Keep GIF under 60 seconds; add short on-screen labels if possible.
