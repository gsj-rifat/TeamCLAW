1. In Slack: Slash command (/report)

What it does: Parses your command text and posts the formatted report in the channel.

How to use:
    /report daily
    /report daily 2025-09-01
    /report weekly
    /report weekly 2025-09-01
    Optional: add “to here” (behavior is already “here” by default with slash commands).

Notes:
    The bot must be invited to the channel.
    This aggregates all insights (no per-channel filter yet via commands).
    You’ll see an ephemeral confirmation; the report posts in the channel.

#################################################################

2. In Slack: Plain text trigger in a channel

What it does: Intercepts messages starting with “report …” or “@BotName report …” and posts a report.

How to use:
    report daily
    report daily 2025-09-01
    @YourBot report weekly
    @YourBot report weekly 2025-09-01

Notes:
    Same behavior as the slash command handler (aggregates all insights).
    Useful if you don’t want to use a slash command.

#################################################################

3. REST API: Daily endpoint

URL: GET /reports/daily

Query parameters:
    date=YYYY-MM-DD (UTC; optional; defaults to today)
    channel_id=C12345678 (optional; filters aggregation to a Slack channel)
    post=true|false (optional; default false; when true it posts to Slack)
    target_channel_id=C12345678 (optional; overrides default post channel)

Examples:
    curl -fsS "https://YOUR_DOMAIN/reports/daily"
    curl -fsS "https://YOUR_DOMAIN/reports/daily?date=2025-09-01"
    curl -fsS "https://YOUR_DOMAIN/reports/daily?channel_id=C012AB3CDE"
    curl -fsS "https://YOUR_DOMAIN/reports/daily?post=true"
    curl -fsS "https://YOUR_DOMAIN/reports/daily?post=true&target_channel_id=C012AB3CDE"

Response includes: period, date, counts, posted flag, target_channel, and report_text.

REST API: Weekly endpoint

URL: GET /reports/weekly

Query parameters:
    week_start=YYYY-MM-DD (UTC Monday; optional; defaults to current week’s Monday)
    channel_id=C12345678 (optional; filters aggregation to a Slack channel)
    post=true|false (optional; default false)
    target_channel_id=C12345678 (optional; override)

Examples:
    curl -fsS "https://YOUR_DOMAIN/reports/weekly"
    curl -fsS "https://YOUR_DOMAIN/reports/weekly?week_start=2025-09-01"
    curl -fsS "https://YOUR_DOMAIN/reports/weekly?channel_id=C012AB3CDE"
    curl -fsS "https://YOUR_DOMAIN/reports/weekly?post=true&target_channel_id=C012AB3CDE"

Response includes: period, week_start, counts, posted flag, target_channel, and report_text
