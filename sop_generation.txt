Generate SOPs
A. Slack commands

Syntax:
    /sop [--days N] [to here]
    In-channel mention variants like: sop [--days N] [to here]

Parameters:
    : topic to generate the SOP for (required)
    --days N: number of past days of context to pull (optional, default configured)
    to here: limits or adjusts collection scope as supported by your handler (optional)

Behavior:
If readiness-first is enabled, the system checks readiness. If complete, it drafts and posts the SOP. If incomplete, it posts a clarification prompt listing missing items and does not draft yet.
If readiness-first is disabled, it attempts to generate directly from available context.

Example:
/sop onboarding flow
/sop deploy to staging --days 30
